# Mechanics Schemas

Use for files in `data/cobblemon/mechanics/*.json`.

Schemas:
- `aprijuices.schema.json` -> `aprijuices.json`
- `berries.schema.json` -> `berries.json`
- `potions.schema.json` -> `potions.json`
- `remedies.schema.json` -> `remedies.json`
- `slowpoke_tails.schema.json` -> `slowpoke_tails.json`

Common pitfalls:
- Putting mechanic keys in the wrong file.
- Using non-expression values where mechanics expect MoLang-like expressions.
