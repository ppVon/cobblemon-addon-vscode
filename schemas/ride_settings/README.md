# Ride Settings Schemas

Use for files in `data/cobblemon/ride_settings/*.json`.

Per-file schemas:
- `bird.schema.json`
- `boat.schema.json`
- `burst.schema.json`
- `dolphin.schema.json`
- `glider.schema.json`
- `helicopter.schema.json`
- `horse.schema.json`
- `hover.schema.json`
- `jet.schema.json`
- `minekart.schema.json`
- `rocket.schema.json`
- `submarine.schema.json`
- `vehicle.schema.json`

Notes:
- Most fields are MoLang/expression strings.
- `burst` currently uses numeric `dashSpeed`.
